# Docker Compose란?

Docker Compose는 서비스 구조를 구성하기 위해 필요한 여러 docker 명령어들을 파일로 관리하는 방법이다. docker compose를 사용함으로써 서비스 구동 환경을 항상 일관되게 유지할 수 있다.

docker inspect

docker-compose up -d —build

docker-compose down
